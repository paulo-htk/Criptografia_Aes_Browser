Crypto AES em Javascript no Navegador

Este é um exemplo simples de uso da Crypto API para criptografia e descriptografia AES.

Atenção: quando acionar a Encriptação, o resultado será escrito na caixa de texto de Decriptação, apagando os dados anteriores. O mesmo acontece com a Decriptação.
